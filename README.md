## Description

This repository contains a collection of spectral, wavelet, and correlation analysis tools commonly applied in geophysical research. The implemented methods include:

1. Coherence and phase-lag analysis
2. Wavelet cross-correlation
3. Wavelet analysis
4. Fourier spectral analysis, including uncertainty estimation using bootstrap resampling

The theoretical background, implementation details, and applications of these methods are described in Puente-Borque et al. (2025) and the references therein.
